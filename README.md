## 💻 Getting started

### Requirements

- [Node.js](https://nodejs.org/en/)
- [pnpm](https://pnpm.io)

**Clone the project and access the folder**

```bash
git clone https://github.com/luancardosoti/ignite-reactjs-04-pizza-shop.git && cd pizza-shop
```

**Follow the steps below**

```bash
# Install the dependencies
$ pnpm i

# Run project
$ pnpm run dev
```
